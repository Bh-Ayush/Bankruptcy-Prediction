# References & Citations

## Datasets

1. **Polish Companies Bankruptcy Dataset**  
   - Source: UCI Machine Learning Repository  
   - Citation: Zikeba, M., Tomczak, S.K., & Tomczak, J.M. (2016). *Ensemble Boosted Trees with Synthetic Features Generation for Bankruptcy Prediction*. Expert Systems with Applications, 58, 93–101.  
   - Link: [UCI Polish Companies Bankruptcy Data](https://archive.ics.uci.edu/ml/datasets/Polish+companies+bankruptcy+data)

2. **Taiwanese Bankruptcy Prediction Dataset**  
   - Source: UCI Machine Learning Repository / Kaggle  
   - Citation: Yeh, I.C., & Lien, C.H. (2009). *The Comparisons of Data Mining Techniques for the Predictive Accuracy of Probability of Default of Credit Card Clients*. Expert Systems with Applications, 36(2), 2473–2480.  
   - Link: [UCI Taiwanese Bankruptcy Prediction](https://archive.ics.uci.edu/ml/datasets/Taiwanese+Bankruptcy+Prediction)  
   - Kaggle mirror: [Kaggle Dataset](https://www.kaggle.com/datasets/fedesoriano/company-bankruptcy-prediction)

3. **US Companies Bankruptcy Dataset**  
   - Source: Kaggle  
   - Citation: Kaggle Dataset. *American Companies Bankruptcy Prediction Dataset* (2019).  
   - Link: [Kaggle Dataset](https://www.kaggle.com/datasets/utkarshx27/american-companies-bankruptcy-prediction-dataset)
